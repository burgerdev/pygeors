\subsection{Detailed Description}
geographic location object 

This object represents a geographic location in terms of zip codes and city limits (as opposed to street addresses and the like). This object prefers German locations at the moment.\begin{DoxyVerb} Basic Usage (and doctests):

>>> g = GeoLoc()
>>> g.zipcode = "87527"
>>> g.complete()
>>> g.city
'Sonthofen'

>>> g = GeoLoc()
>>> g.city = "Sonthofen"
>>> g.complete()
>>> g.zipcode
'87527'

>>> g = GeoLoc()
>>> g.latlon = (47.51, 10.29)
>>> g.complete()
>>> g.zipcode
'87527'
>>> g.city
'Sonthofen'
\end{DoxyVerb}
